## 🔧 حل مشاكل التشغيل | Troubleshooting Launch Issues

### ❌ المشكلة: النظام لا يفتح تلقائياً
**الحلول بالترتيب:**

#### 1️⃣ الحل الأول | First Solution
```
انقر بالزر الأيمن على الملف HTML
اختر "فتح باستخدام"
اختر أي متصفح متاح
```

#### 2️⃣ الحل الثاني | Second Solution
```
اسحب الملف HTML إلى نافذة متصفح مفتوحة
```

#### 3️⃣ الحل الثالث | Third Solution
```
انسخ مسار الملف والصقه في شريط عنوان المتصفح
```

#### 4️⃣ الحل الرابع | Fourth Solution
```
افتح متصفح → Ctrl+O → اختر الملف HTML
```

### ❌ المشكلة: تحليل الملفات لا يعمل (النظام المحسن)
**الحلول:**

1. **تأكد من الاتصال بالإنترنت** لتحميل مكتبات التحليل
2. **انتظر قليلاً** لتحميل المكتبات (قد يستغرق دقيقة)
3. **أعد تحميل الصفحة** (F5 أو Ctrl+R)
4. **جرب متصفح Chrome** للأداء الأفضل
5. **استخدم النظام المحمول** كبديل

### ❌ المشكلة: البيانات لا تُحفظ
**الحلول:**

1. **لا تستخدم وضع التصفح الخاص** (Incognito/Private)
2. **اسمح للمتصفح بحفظ البيانات** المحلية
3. **امسح ذاكرة التخزين المؤقت** وأعد المحاولة
4. **تأكد من مساحة التخزين** المتاحة

## 🎯 كيفية الاستخدام | How to Use

### 🔐 تسجيل الدخول | Login
1. **افتح النظام** بأي من الطرق أعلاه
2. **البيانات معبأة مسبقاً** في النموذج
3. **انقر "تسجيل الدخول"** للدخول مباشرة

### 🌐 تغيير اللغة | Language Change
- **زر اللغة** في الزاوية العلوية
- **تبديل فوري** بين العربية والإنجليزية
- **حفظ تفضيل اللغة** تلقائياً

### 👨‍🎓 إدارة الطلاب | Student Management
1. **انتقل لتبويب "الطلاب"**
2. **املأ بيانات الطالب** (عربي وإنجليزي)
3. **اسحب ملفات الطالب** للرفع
4. **انقر "إضافة الطالب"** للحفظ

### 🔍 تحليل الملفات | File Analysis (النظام المحسن فقط)
1. **انتقل لتبويب "تحليل الملفات"**
2. **اختر نوع التحليل** (صور، PDF، مستندات)
3. **اسحب الملفات** إلى منطقة الرفع
4. **انتظر التحليل** (يظهر مؤشر التقدم)
5. **اعرض النتائج** في الأسفل

### 📊 مراقبة الإحصائيات | Monitor Statistics
- **تحديث تلقائي** للأرقام
- **رسوم بيانية تفاعلية** (النظام المحسن)
