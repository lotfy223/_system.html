üìä Data Science Visualizations

üî• Correlation Heatmap

Shows correlations between variables in the penguins dataset. Darker colors indicate stronger correlations.

import seaborn as sns import matplotlib.pyplot as plt import pandas as pd penguins = pd.read_csv('penguins.csv') corr = penguins.corr() sns.heatmap(corr, annot=True) plt.show()

üéØ PCA Dimensionality Reduction

Principal Component Analysis reduces data dimensions while preserving most variance.

from sklearn.decomposition import PCA penguins = penguins.dropna() X = penguins.drop('Species', axis=1) pca = PCA(n_components=2) X_pca = pca.fit_transform(X)

üì¶ Box Plot & Histogram

Box plots show data distribution including median, quartiles, and outliers.

heights = [63, 64, 66, 67, 68, 69, 71, 72, 73, 55, 75] plt.figure(figsize=(12, 5)) plt.subplot(1, 2, 1) plt.boxplot(heights) plt.subplot(1, 2, 2) plt.hist(heights, bins=8)
